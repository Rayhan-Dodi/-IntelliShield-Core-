#### Video Demo: <URL HERE>  
#### Description:  
IntelliShield Core is a machine learning-powered Chrome extension designed to detect and prevent access to malicious websites in real time. It acts as a security layer between the user and the web by analyzing website behavior and classifying pages as safe or potentially harmful. Unlike traditional blacklist-based systems, it uses machine learning to identify both known and emerging threats dynamically, providing adaptive protection against phishing and malicious web activity.
